# 💎 Diamond Price Prediction App

An end-to-end Machine Learning project that predicts diamond prices based on physical attributes and quality features using K-Nearest Neighbors (KNN) Regression and Streamlit.

## 📌 Project Overview
- **Objective:** Build a predictive model to estimate diamond prices in USD.
- **Model:** K-Nearest Neighbors (KNN) Regressor.
- **Preprocessing:** `ColumnTransformer` with `StandardScaler` for numerical attributes and `OrdinalEncoder` for categorical traits (`cut`, `color`, `clarity`).
- **Deployment:** Interactive web interface powered by Streamlit.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn, Streamlit, Pickle
- **Environment:** Jupyter Notebook
